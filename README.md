<h1 align="center">gnib-wallpaper</h1>

<p align="center">每天看一張別的地方的風景.</p>

---

Bing 在各國有自己的「今日桌布」, 不同國家挑的圖不一樣. 同一天, 美國可能是落磯山的清晨, 日本可能是某座神社的櫻花, 法國可能是普羅旺斯的薰衣草. 這個小程式把這些圖收在一起, 讓你隨手翻看, 看到喜歡的, 按一下就直接換成自己的桌面.

## Features

- 打開以後, 上方一排國旗代表各國的今日桌布. 點哪面旗子就跳到哪一國, 圖片是 4K UHD.
- 可使用工具列上的設為桌布按鈕將 Windows 桌面設定成當前觀看圖片
- 透過設定指定國家, 下次啟動就直接停在那一國, 不用每次手動切, 並且當該國的今日桌布更新時會自動替換桌布.
- 透過設定可以調整自動更新桌布的頻率.
- 系統匣功能, 可縮小到右下角通知區, 不佔工作列空間, 也不會跳通知打擾.
